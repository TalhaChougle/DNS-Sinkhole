What is this dashboard showing me?
Every time you visit a website, your computer first asks "what's the IP address for this site?" — that's called a DNS query. Your sinkhole intercepts every single one of those questions. If the site is on the blocklist (ads, trackers, malware), it returns a fake IP so your computer never connects. If it's safe, it gets the real IP and loads normally. Everything is logged with the exact date and time — go to the Query Logs tab to see it.
