What is this dashboard showing me?
Every time you visit a website, your computer first asks "what's the IP address for this site?" — that's called a DNS query. Your sinkhole intercepts every single one. If the site is on the blocklist (ads, trackers, malware), it returns a fake IP so your computer never connects. If it's safe, the real IP is returned normally. Everything is logged with the exact date and time.
